墨光 (MarkLight) 是一款为开发者与创作者设计的 Markdown 编辑器。基于自研内核，提供真正的所见即所得体验与极致的性能表现。

下载正式版 v0.2.0 查看源代码

为什么要选择墨光？

🚀

所见即所得

基于 ProseMirror 自研内核，实时渲染你的每一行输入。告别传统的左右分栏，享受沉浸式写作体验。

📦

本地优先

支持图片自动本地化存入 assets 目录。所有数据均存储在你的磁盘上，隐私且安全。

📱

微信排版

内置多套精美排版主题，支持带行内样式的 HTML 导出，一键复制，完美适配公众号发布。

📊

专业渲染

完美支持 KaTeX 数学公式（带预览气泡）与 Mermaid 工业级图表，满足学术与技术写作需求。

🖥️

原生体验

针对 Windows 与 macOS 深度适配。支持多窗口、命令面板与原生快捷键，体验如丝般顺滑。

⌨️

效率面板

强大的命令面板 (Cmd+K) 让你无需离开键盘即可完成文件搜索、功能切换等所有复杂操作。

硬核技术，稳定可靠

Rust & Tauri 2.0 ProseMirror Core Vue 3 & TS Tailwind CSS 4.0
