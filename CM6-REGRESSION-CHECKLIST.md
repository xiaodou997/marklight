# CM6 回归验收清单（CM6-only）

> 目的：确保 CM6 单引擎编辑流程稳定。

## 1. 基础编辑
- [ ] 新建文档可输入/删除文本，保存后内容一致
- [ ] 撤销/重做连续操作正确（含跨行编辑）
- [ ] 光标上下左右移动无跳跃、无“选中蓝框”异常
- [ ] 选区跨多行行为正常（复制/替换）

## 2. Live Preview 与源码显隐
- [ ] 标题行：非光标行隐藏 `#`，光标进入时显示并可逐字符编辑
- [ ] 引用行：`>` 前缀显隐逻辑正确
- [ ] 列表行：`-` / `*` / `+` / `1.` 原始符号保留
- [ ] 任务列表：`- [ ]` / `- [x]` 可视化与源码切换一致
- [ ] 行内语法：粗体/删除线/代码/链接在非光标行正确渲染

## 3. 高级块渲染
- [ ] 代码块：非光标行显示 widget，进入后回到源码编辑
- [ ] 数学块：`$$...$$` 非光标行渲染，光标进入可编辑源码
- [ ] 表格：非光标行渲染为表格，光标进入可编辑原始 Markdown

## 4. 交互功能
- [ ] 搜索/替换（大小写、上一个/下一个、替换、全部替换）
- [ ] 快捷键（加粗/斜体/标题/列表/撤销重做）行为符合设置
- [ ] 气泡菜单（bold/italic/code/link/h1/h2）可用
- [ ] 智能粘贴（富文本 -> Markdown）可用
- [ ] 图片拖拽与粘贴均可插入，且不重复插入

## 5. 导出与文件链路
- [ ] HTML 导出内容正确
- [ ] PDF 导出内容正确
- [ ] 微信复制/导出样式正确
- [ ] 自动保存触发与状态提示正常

## 6. 下线确认
- [ ] 设置中不再显示编辑器引擎切换项
- [ ] 菜单命令（查找/替换/命令面板）在 CM6 正常可用
- [ ] 项目启动与设置加载过程无异常日志
