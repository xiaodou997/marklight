# MarkLight 开发路线图

## 项目定位

MarkLight 是一款跨平台 Markdown 编辑器，基于 Tauri 2.x + Vue 3 + ProseMirror 构建，专注于提供流畅的写作体验。

---

## 核心功能完善

### 1. 快捷键体系 ✅ (已完成)

- [x] 跨平台快捷键支持（Mac: Cmd, Win/Linux: Ctrl）
- [x] 格式化快捷键（加粗、斜体、删除线、高亮、行内代码）
- [x] 标题快捷键（Ctrl+1~6, Ctrl+0 段落）
- [x] 列表快捷键（有序、无序、任务列表）
- [x] 块级元素快捷键（代码块、引用）
- [x] 撤销/重做快捷键
- [x] 设置中的快捷键说明页面
- [ ] 快捷键自定义功能

### 2. 代码块增强 (计划中)

- [ ] 代码块语言选择下拉菜单
- [ ] 代码块行号显示
- [ ] 代码块复制按钮
- [ ] 代码块语法高亮主题切换
- [ ] 代码块折叠功能

### 3. 表格编辑优化 (计划中)

- [ ] 表格工具栏增强
- [ ] 表格单元格快捷导航
- [ ] 表格排序功能
- [ ] 表格导出为 CSV
- [ ] 复杂表格支持（合并单元格）

### 4. 图片管理 (计划中)

- [ ] 图片本地存储管理
- [ ] 图片上传到图床
- [ ] 图片拖拽排序
- [ ] 图片批量导出
- [ ] 图片压缩优化
