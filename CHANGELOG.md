# 更新日志 (Changelog)

所有对 MarkLight 的重要更改都将记录在此文件中。

## [1.0.0] - 2026-04-19

### 🎉 首个正式稳定版

经过多次迭代，MarkLight 正式发布 1.0.0 版本，实现 Typora 风格的所见即所得 Markdown 编辑体验。

### ✨ 核心功能（已完成）
- **所见即所得编辑**：基于 TipTap (ProseMirror)，代码块、表格、公式等可直接在渲染态编辑
- **完整 Markdown 支持**：标题、列表、引用、任务列表、代码块、表格、图片、水平线
- **行内格式**：粗体、斜体、删除线、行内代码、高亮、链接、上标、下标
- **数学公式**：KaTeX 渲染，点击进入 LaTeX 编辑
- **Mermaid 图表**：渲染流程图/时序图，点击编辑源码
- **Callout 块**：Obsidian 风格提示框
- **Frontmatter**：YAML 元数据面板
- **Wikilink**：`[[page]]` / `[[page|alias]]` 支持
- **Typora 风格源码提示**：光标进入格式化文本时显示 Markdown 语法标记
- **斜杠命令**：`/` 菜单快速插入块
- **拖拽排序**：块级节点拖拽重排

### 🛠️ 应用功能
- 文件树侧边栏、大纲导航、命令面板 (Cmd+K)
- 焦点写作模式、自动保存、搜索替换
- 图片本地化存储、全屏预览
- 微信公众号格式导出（多主题）
- 多窗口独立编辑
- 深色/浅色主题

### 🔧 技术架构
- 编辑器从 CodeMirror 6 迁移至 TipTap (ProseMirror)
- Mark Token 实体化方案实现完美 round-trip
- 完整测试覆盖（30+ round-trip 测试用例）

## [0.3.0] - 2026-04-10

### 🚀 编辑器全面重构
- **编辑器引擎迁移**：从 CodeMirror 6 迁移至 TipTap (ProseMirror)，实现 Typora 风格的所见即所得编辑。
- **代码块可编辑**：代码块始终渲染语法高亮，可直接在渲染态编辑代码内容（基于 CodeBlockLowlight）。
- **表格可编辑**：表格始终渲染为表格样式，可直接编辑单元格内容，支持 Tab 导航。
- **数学公式**：支持 `$$` 块级公式（KaTeX 渲染），点击进入 LaTeX 编辑。
- **Mermaid 图表**：渲染 Mermaid 流程图/时序图，点击进入源码编辑。
- **Callout 块**：支持 Obsidian 风格的提示框（`> [!NOTE]` 等）。
- **Frontmatter**：YAML 元数据面板，可折叠/展开，双击编辑。
- **搜索替换增强**：基于 ProseMirror 文档模型的精确匹配和定位。
- **Markdown InputRules**：输入 `$$` 或 ` ```mermaid ` 自动转换为对应块。
- **Wikilink**：支持 `[[page]]` 和 `[[page|alias]]` 语法，渲染为内联链接。
- **上标/下标**：支持 `^sup^` 上标和 `~sub~` 下标语法。
- **内联源码提示**：Typora 风格，光标进入格式化文本时显示 Markdown 语法标记。
- **斜杠命令**：输入 `/` 弹出命令菜单，快速插入标题、列表、代码块、表格、公式等块。
- **拖拽排序**：块级节点左侧显示拖拽手柄，支持拖拽重新排列块顺序。

### 🧹 工程清理
- 移除全部 CodeMirror 6 依赖和代码。
- 移除过期的 CM6 迁移文档。
- 新增 ARCHITECTURE.md 技术方案文档。
- 更新 ROADMAP.md 路线图。

## [0.2.2] - 2026-03-20

### 🚀 核心调整
- **编辑器统一**：正式下线 ProseMirror，实现 CodeMirror 6 单引擎运行。
- **依赖清理**：移除全部 `prosemirror-*` 依赖及 `@types/prosemirror-tables`。
- **导出链路**：微信/HTML 导出统一为 Markdown 直出，不再依赖 PM 文档树。

### 🧹 工程清理
- 删除 PM 编辑器实现与相关插件目录。
- 设置页移除编辑器引擎切换能力，统一展示为 CM6。
- 更新 CM6 文档为 CM6-only 口径，并清理过期迁移文档。

## [0.2.1] - 2026-03-06

### 🚀 新功能
- **图片预览重构**：侧边栏图片现在默认在主区域预览，双击可进入全屏毛玻璃模式。
- **Gitee 集成**：增加了 Gitee 项目主页入口，并配置了自动化镜像同步。
- **帮助菜单**：在原生菜单栏中增加了项目文档、问题反馈及检查更新入口。

### 🎨 UI 与样式
- **沉浸式标题栏**：为 Windows 和 Linux 实现了全新的自定义菜单标题栏，视觉与 macOS 对齐。
- **品牌更新**：正式更名为“墨光 (MarkLight)”，更新了开发者署名为个人 ID。

### 🐛 修复与适配
- **数学公式优化**：修复了退格键删除整个公式的问题，支持点击精准定位光标及渲染失败回退。
- **代码高亮修复**：解决了代码块中 `__name__` 等特殊变量变白的问题，并强化了样式隔离。
- **编译修复**：解决了 Tauri v2 下 `notify` 依赖缺失及类型推导导致的构建失败问题。

## [0.2.0] - 2026-03-06

### 🚀 新功能
- **核心内核**：基于 ProseMirror 的自研所见即所得 (WYSIWYG) 编辑内核。
- **自动保存**：支持可配置间隔的后台防抖保存。
- **搜索替换**：完整的文档内搜索与替换功能，支持大小写敏感及快捷键。
- **图片本地化**：粘贴或拖拽图片时自动存入项目 `assets/` 目录并使用相对路径。
- **多窗口支持**：支持通过快捷键或文件树 Cmd+点击打开多个独立编辑窗口。
- **命令面板**：`Cmd+K` 快速唤起，支持命令执行与文件搜索。
- **数学公式**：增强的 KaTeX 预览气泡，光标移入即刻实时渲染。
- **文件管理**：文件树右键菜单支持新建、重命名、删除（进入回收站）及在系统文件管理器中显示。

### 🎨 UI 与样式
- **沉浸式标题栏**：为 Windows 和 Linux 实现自定义无边框标题栏，视觉风格与 macOS 统一。
- **滚动条美化**：重新设计了全局滚动条，适配极简白设计风格。
- **加载状态**：为超长文档和文件切换增加了沉浸式 Loading 遮罩。

### 🐛 修复与适配
- **解析器稳定性**：修复了 Markdown 中包含 HTML 标签导致解析崩溃的问题。
- **跨平台快捷键**：抽象了 `isModKey` 逻辑，确保 `Cmd` (Mac) 与 `Ctrl` (Win/Linux) 体验一致。
- **Linux 兼容性**：适配了 Linux 下的文件管理器打开指令。
- **系统集成**：汉化并精简了原生菜单栏。

### 🏗️ 维护与工程
- **CI/CD**：配置 GitHub Actions 自动构建 macOS (dmg) 和 Windows (nsis exe) 安装包。
- **文件关联**：在系统层建立 `.md` 文件关联。
